# Santiago Sanchez — Portfolio

Personal portfolio website for a QA Manual & Automation Engineer, built with Astro.

**Live demo:** https://santisanchez4.github.io/portfolio

## Tech stack

- **Astro** — static site framework
- **CSS** — custom design system (no UI libraries)
- **GitHub Pages** — deployment

## Run locally

```bash
npm install
npm run dev
```

The site will be available at `http://localhost:4321`.
